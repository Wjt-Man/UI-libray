### hacker-fast-ui 组件库

### 快速开始

#### 1.安装组件库

```bash
npm i hacker-fast-ui
```

#### 2.引用组件库

```js
//全部引入
import "hacker-fast-ui/dist/css/index.css";
import MUI from "hacker-fast-ui";
Vue.use(MUI);

//按需引用
import "hacker-fast-ui/dist/css/demo.css";
import { Demo } from "hacker-fast-ui";
Vue.use(Demo);
```
